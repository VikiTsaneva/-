# Parki
### Сайтът е активен на https://parki-app1.vercel.app/
Parki е умно приложение за паркиране, което в реално време показва свободните места на паркинги и улици. С помощта на интелигентни алгоритми и актуални данни, Parkly ти помага бързо да намериш място, да спестиш време, гориво и излишно обикаляне из града.

## Стъпки за стартиране от папката на проекта
1. Стартирай `start.bat`:
   - `start.bat`
2. Отвори браузър и посети: `http://localhost:3000`

> Забележка: `start.bat` проверява Node.js и използва `npx serve .` за стартиране на локален сървър.
